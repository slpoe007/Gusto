# 格时图官网 · 全球推广战略建议

> 基于老网站（gustotechnik.com）的现状分析和新网站的技术架构，以下是面向全球市场的推广策略。

---

## 一、老网站诊断：为什么必须重做

| 问题 | 影响 |
|------|------|
| 部分页面报错"模块不存在" | 潜在客户直接流失 |
| 设计停留在 5 年前（Bootstrap + jQuery） | 在竞品面前显得不专业 |
| 无响应式优化 | 手机端体验差（全球 60%+ 流量来自移动端） |
| 中文内容混乱（Technical页出现电视安装内容） | 品牌形象受损 |
| SEO 几乎为零（meta keywords/description 均为空） | Google 搜不到 |
| 产品仅有车型列表，无产品详情、无品牌介绍 | 无法促成询价 |
| 无赛事内容（核心差异化优势未体现） | 品牌故事缺失 |

## 二、新网站定位

**格时图 ≠ 普通汽车配件店**

核心差异：
- 竞技基因：TCR China 官方底盘&引擎供应商

- 研发能力：GETuned 自主品牌（赛道技术→民用产品）
- 四地运营：覆盖中国主要赛车市场

**一句话定位：** "从赛道到公路 — 中国领先的赛车配件及技术服务商"

**目标受众：**
- B2B：车队经理、改装店主、赛事组织方
- B2C：高性能车车主、赛道日爱好者
- 海外：亚洲/欧洲的赛车行业采购商、品牌方

## 三、技术层面的全球推广基础

### 3.1 SEO 地基（新网站已具备）
- Next.js 16 → 天然支持 SSR/SSG（搜索引擎友好）
- 每个产品/赛事页面独立 URL → 利于收录
- 结构化数据（建议添加）：

```html
<!-- 产品页：Product Schema -->
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Product",
  "name": "ALCON ADVANTAGE EXTREME",
  "brand": { "@type": "Brand", "name": "ALCON" },
  "description": "...",
  "offers": { "@type": "Offer", "price": "28000", "priceCurrency": "CNY" }
}
</script>

<!-- 赛事页：Event Schema -->
<!-- 新闻页：Article Schema -->
<!-- 公司页：Organization Schema -->
```

### 3.2 性能指标目标（Core Web Vitals）
- LCP < 2.5s（图片压缩+CDN）
- FID < 100ms
- CLS < 0.1
- 目标：Google PageSpeed 90+

### 3.3 多语言架构
- 第一阶段：英文为主 + 中文切换（当前设计已包含）
- 第二阶段：日文/韩文（亚洲赛车市场）
- 建议用 `next-intl` 实现 SEO 友好的多语言路由（`/en/products`、`/zh/products`）

## 四、内容策略

### 4.1 产品页面要回答的问题
每个产品页要让潜在客户在 5 秒内知道：
1. 这是什么？（产品名+品牌+品类）
2. 装什么车？（适用车型）
3. 多少钱？（价格区间）
4. 怎么买？（询价按钮/联系方式）

### 4.2 赛事内容的价值
赛事内容不是"装饰"，是**信任背书**：
- 赛报 = "我们不是纸上谈兵，产品在 TCR China 实战验证"
- 车手介绍 = 品牌人格化
- 赛历 = "我们一直在赛道"
- 每篇赛报配 5+ 张高清比赛照片

### 4.3 新闻/博客
- 每月至少 2 篇更新（产品发布/赛事回顾/技术科普）
- 长尾关键词布局：如"TCR赛车刹车选择指南""避震器调校入门"
- 目的：让格时图成为中文+英文赛车配件领域的权威信息源

## 五、询价转化漏斗

```
Google搜索 "TCR brake kit China"
  → 落地产品详情页
    → 查看规格+适用车型
      → 点击"询价"按钮
        → 填写表单（姓名/邮箱/车型/需求）
          → 自动发送邮件至深圳销售团队
            → 24小时内回复报价
```

关键点：
- 询价表单要短（4-5个必填项）
- 表单提交后即时反馈（"已收到，我们将在24小时内回复"）
- 可选：集成 WhatsApp / 微信快捷联系

## 九、立即可做的事（不等市场部资料）

以下工作现在就能推进：

1. **结构化数据模板**：预先写好 Product / Article / Event / Organization 的 Schema JSON-LD，数据一到就填入
2. **SEO Meta 优化**：给每个页面写好 `<title>` + `<meta description>`（中英文）
3. **404 页面**：自定义一个有趣的赛车主题 404 页
4. **性能优化**：图片懒加载、字体子集化、CSS 精简
5. **Sitemap**：配置 `next-sitemap` 自动生成
6. **robots.txt**：正确配置以引导搜索引擎
7. **Open Graph 标签**：确保分享到社媒时显示正确的标题+图片+描述
